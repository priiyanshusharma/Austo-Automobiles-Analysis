# Austo Automobile Analysis

## Overview

This project analyzes customer behavior and vehicle purchasing patterns using data analysis.

## Objective

To understand how demographics, income, and lifestyle factors influence automobile buying decisions.

## Tools Used

Python

Pandas

Matplotlib / Seaborn

## Key Insights

SUVs are preferred by older and high-income customers

Hatchbacks dominate the budget segment

Dual-income households spend more on vehicles

Sedans are popular among salaried professionals

## Files

Austo_Motor_Project.ipynb → *Analysis*

Austo_Motor_BusinessReport.pdf → *Report*

## Conclusion

The project provides insights that can help improve marketing strategies and target the right customer segments.
